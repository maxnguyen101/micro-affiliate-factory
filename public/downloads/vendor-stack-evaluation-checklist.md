# Vendor Stack Evaluation Checklist

Use this during demos and trial reviews.

## Checklist

- [ ] CRM fit: bi-directional sync for owners, stages, notes, activity logs
- [ ] Workflow overhead: admin team can maintain rules without engineering dependency
- [ ] Reporting: role-based adoption and impact views are exportable
- [ ] Security/compliance: consent, retention, and PII controls are documented
- [ ] Migration risk: historical data import/export is validated
- [ ] Support quality: onboarding plan and SLA escalation path are explicit
- [ ] Commercial fit: pricing scales with projected seats and usage
- [ ] Pilot success criteria: go/no-go metrics are defined before contract signature
